# StreamDock

> **Your OBS, finally set up right.**
>
> StreamDock is a desktop companion app for OBS Studio. It auto-configures OBS through a 5-question wizard, saves and switches profiles instantly, and monitors your stream health in real time all without touching a single OBS setting manually.

---
## 1. What StreamDock Does

| Feature | Free | Pro ($7/mo) | Lifetime ($49) |
|---|---|---|---|
| Setup Wizard — all 4 use cases | ✅ | ✅ | ✅ |
| Saved profiles | 2 max | Unlimited | Unlimited |
| Profile export / import | ❌ | ✅ | ✅ |
| Health monitor — basic | ✅ | ✅ | ✅ |
| Health monitor — full (5 metrics + alerts) | ❌ | ✅ | ✅ |
| Auto-fix stream issues | ❌ | ✅ | ✅ |

**The wizard** asks 5 questions (use case, quality, GPU, audio, layout) and auto-creates OBS scenes, scene collections, and audio inputs — all named consistently as `StreamDock_<ProfileName> — <SceneName>`.

**The health monitor** polls OBS every 2 seconds for dropped frames, CPU usage, render lag, disk space, and bitrate. Alerts appear in plain English.
